## 异常分析
- `java.lang.IllegalArgumentException: argument type mismatch`
 
 **关键错误行数**
 
```java
writeMethod.invoke(obj,columnValue);
```
**错误分析**

参数类型不匹配 //要求：实体类的属性和数据库表的列名保持一种
